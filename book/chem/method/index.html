非常有用的: 做题方法汇总

总的道理: 性质!

性质

学好高中化学, 无非就是: 性质

要清楚所有常见物质, 有什么性质

然后也要能列举出来: 每个性质有哪些物质拥有

(比如: 练习一下, 列举出常见的氧化性与还原性物质?)

然后, 做题的时候思考: 物质的各个性质之间, 有没有相互作用的?

然后你就会知道, 发生了什么, 从而解释一些现象

大题的答题方法:

各类题型:

  • 工业流程: 每个箭头标成分!

  • 写方程式:

    • 需要先判断的: 生成啥? 是酸是碱? 变不变价?
    • 然后, 如果变价, 用氧化还原那一套配平; 不变的话, 直接原子守恒配平
    • 配完简单检验一下, 至少不要出现很明显的低级错误, 比如两边都有 \(\ce{H_2O}\) 这种

  • 描述机理:

    • 多用 专业术语. 类似 "活性点位", "吸附", "转化率/选择性", ...

  • 归因解释:

    • 两大角度: 速率！ 平衡！ 开始回答前, 先明确: 是速率, 还是平衡?
    • 枚举性质, 找到性质之间的关联!

  • 实验题:

    • 注意表述. 如 "边搅拌边加入", "逐滴滴加", "取最后一次洗涤滤液"...
    • 注意 “终点”：做到什么时候为止？比如 “至不再产生沉淀”，“至固体不再溶解（浸取时）”，“至滴加xxx后不再产生沉淀（洗涤）”

具体问题:

  • "为何选用xxx (特定数值)?" :
    若过高, ...; 若过低, ...;

  • "xx为何上升/下降?" 且原因不单一 (以上升为例):
    "...导致上升, ...导致下降, 且前者影响大于后者, 所以上升"
    一定要提到 对立的两面, 并且说明 谁是更主要的

  • 关于催化剂:
    "活性点位": 反应的关键位置, 吸附要反应的物质, 并且在这个地方反应
    活性点位能吸附一类物质的时候, 可能类似结构的也能吸附, 就要考虑 竞争吸附 的问题
    催化剂会 失活, 可能原因: 温度太高; 固体覆盖 (覆盖活性点位); ...
    对于有机反应, 常见的情况是: 积碳

反应速率的影响因素

  • 浓度，温度，催化剂 （压强：本质也是浓度）
  • 温度的副作用：受热分解/挥发，气体溶解度下降，催化剂活性降低
  • 接触面积，接触的有多充分 （打成粉末，疏松多孔，逆流接触，减缓通气）
  • 搅拌 （通入气体也有搅拌作用）
  • 原电池
  • 副反应：竞争阻碍主反应，或者产物会带来其它副作用（比如固体覆盖）
  • 自催化：反应的产物可作为催化剂催化原反应，如 \(\ce{KMnO_4 + H_2C_2O_4}\)
  • 固体：覆盖，阻碍反应 （有机：积碳）
  • 配位：“缓释”离子，降低速率

应用：提高浸取率的手段 （5条）

  1. 增大浓度
  2. 适当 升温
  3. 粉末
  4. 搅拌 （或加快搅拌速率）
  5. 延长浸取时间

实验操作表达

  • 加入："边搅拌边加入"（常驻）， "逐滴滴加" （需要“慢”时）

  • 过滤：若有后续操作，说明：是对滤液还是滤渣操作？表达成（比如）
    (a) “过滤，取滤液，加入 xxx，继续操作”
    (b) “过滤，取滤渣，洗涤2-3次，干燥”

  • 过滤：有很多种形式
    趁热过滤：比如有时候温度冷却下来溶质就会析出，但此时还不希望它出来，就需要趁热过滤
    抽滤：相当于在另一头去 “吸” 滤液，作用：(1) 加快过滤速率 (2) 使滤渣更干燥
    砂芯漏斗：当滤液或滤渣有强腐蚀性时，滤纸无法承担，就用这个过滤

  • 有 “特殊形式” 的操作：
    真空干燥：若固体易被氧化 / 易潮解，“空气不耐受”，则真空干燥
    减压蒸发：降低沸点，可以在更低温度下蒸发
    乙醇洗涤：可以减少溶解，并且乙醇挥发可以带走水分 促进干燥
    饱和溶液洗涤：也是可以避免产物被溶解的办法
    冰水洗涤：通过降低温度，减少溶解，适用于温度低是溶解度明显低的
    水浴加热：优点：受热更均匀，温度更可控 （类似的还有油浴，优点类似但温度更高）

  • 检验洗涤干净："取最后一次洗涤滤液少量于试管中，加入 xxx （检验），无现象"
    该 “检验” ，通常是检验 \(SO_4^{2-}\) 和 \(Cl^-\)，参考对应离子的检验方法加入试剂

  • “终点”：做到什么时候为止？
    比如 “至不再产生沉淀”，“至固体不再溶解（浸取时）”，“至滴加xxx后不再产生沉淀（洗涤）”
