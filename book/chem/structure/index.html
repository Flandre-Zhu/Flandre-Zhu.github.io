物质结构

原子

原子：原子核，核外电子 （同物理）

电子的分布：电子云

分级制度：层 与 级

  • 能层：就是平常说的 “层”，“一圈” 是 “一层”
  • 能级：同一个能层里可以继续细分出能级，比如 \(2\) 层的 \(2s,2p\)

不同能级，其电子云形状不同，该形状叫做 轨道，有 \(s,p,d,f\) 几种

  • 形状：\(s\) 球形，\(p\) 纺锤形
  • 能量： 同一能层：\(s<p<d<f\) 不同能层：参考 “圣诞树”
  • 伸展方向（种类数）：\(1,3,5,7\)
  • 电子数量：\(2,6,10,14\) 是上述两倍，是因为，同一个轨道里，最多容纳两个电子，自旋为 \(\uparrow \downarrow\) （泡利不相容原理）

对于一个电子，其属性有：

  • 层：在哪一层
  • 形：轨道形状，\(spdf\)
  • 伸：哪一种伸展
  • 旋：哪一种自旋

核外电子，排布方式：

  • 三大规则（能量低，泡利不相容，洪特 “分散” “同旋”）
  • 特殊情况 （全空，半满，全满）

不同轨道会 “融合”，混到一起，形成 杂化轨道

  • 需掌握的杂化：\(sp,sp^2,sp^3\)
  • 需要知道他们的形态，以及判断
  • 还有别的杂化，不需要掌握

  • 原子结构示意图？
  • 电子式？
  • 电子排布式：核外？外围？“简式”？
  • 轨道排布式 / 轨道表示式？

化学键，分子间作用力

“键”：的本质是 力，是化学微粒（离子，原子，分子）间的相互作用力

细节：“键” 形成的形状并非固定，而是动态平衡，平常说的形状是指 平衡位置 形成的性质，它实际上是会振动的

金属键

金属中自由移动电子，与金属阳离子，形成

带来性质：导电性，导热性，延展性

强弱：

  • 原子化热 （变成气态原子需要吸收的能量）越大，则金属键越强
  • 半径小，自由电子“浓度”大，则金属键越强

金属键强，则：溶沸点高，硬

离子键

正负离子之间形成

无方向，不会饱和

强弱：

  • 晶格能 （拆开离子晶体使其变成气态原子需要吸收的能量）越大，则离子键越强
  • 半径小，电荷多，则离子键越强

离子键强，则：溶沸点高

共价键

形成共用电子对

饱和性，方向性 ：有特定的键角，元素成键数量有限 （参考有机）

分为：\(\sigma\) 键（sigma），\(\pi\) 键

  • \(\sigma\) 能旋转，\(\pi\) 不能
  • “双键”，“三键” 中：一个是 \(\sigma\)，剩下都是 \(\pi\)
  • 通常 \(\sigma\) 更稳定

分为：极性键，非极性键

  • 极性来自 电负性 的差异
  • 差的越大，极性越大

分子：极性分子，非极性分子

  • 极性键也可以因为对称性形成非极性分子 例如 \(CCl_4\) 分子
  • 相似相溶 原理

共价键的强弱：键能

  • 键能越大，越难断裂，越稳定

  • 键能与 \(\Delta H\)：见选必一

特殊的共价键：配位键

  • 一般共价键共享电子，是 1+1，“AA制”
  • 配位键则是 0+2，“请客”
  • “0”：空轨道，“2”：孤电子对
  • 空轨道的是 配位中心，孤电子对的是 配体

配位键的表示：由配体指向配为中心，例如 \(NH_3 \to Cu^{2+}\)

形成配位键后，性质 和共价键完全一样，比如 \(NH_4^+\) 分子中四个 \(N-H\) 键是 完全一样 的，即使其中是 3 个普通共价键和一个配位键，但他们没有区别

配位键形成的物质：配合物

应用：

  • 首先要知道它存在，认识常见物质，比如铜氨离子，银氨离子，EDTA 链接：生物，血红素中 \(Fe^{2+}\)，叶绿素中 \(Mg^{2+}\)，都是配位
  • 配合物有 吸收金属离子 的作用，可以： 降低金属离子浓度，用于浓度调控 “遮蔽” 其反应，用于保护或者避免副反应
  • 配位的反应是 可逆反应，有 配位平衡

分子间作用力

广义来说，它也是 “键”，但 不属于化学键

常见的：范德华力，氢键

范德华力：普遍存在，弱

  • 大小：相对分子质量越大，作用力越大

氢键：重要！

形成原理：\(HO \cdots H-X\)

  • 这里 \(O\) 带负电，假如 \(H-X\) 中的 \(H\) 带着一定量的正电 （取决于电负性差），就会形成氢键
  • 这个 \(O\) 也可以换成 \(N\) 这种强电负性的非金属
  • 常见的：水分子，含 \(-OH\) 的分子（比如乙醇，乙酸之类的），\(NH_3\) 与含氨基的物质 比如，DNA 的碱基对，蛋白质中的一些氨基酸，也会形成氢键，主要是靠 \(N\) 元素

氢键的影响：

  • 分子间氢键：大幅增加 溶沸点
  • 分子 内 氢键：降低 溶沸点
  • 溶解度：若溶质和溶剂之间形成氢键，有利于溶解 例如：水和乙醇
  • 其它：增大水的比热容，使得水有流动性，使得冰的密度小于水 ... 有利于生命的活动

晶体

分类

根据化学键，分为：金属晶体，离子晶体，共价晶体，分子晶体

特殊注意：

  • 共价晶体意味着：整个晶体里面全都是共价键连接，没有单独的分子出现 例如：金刚石，硅，碳化硅，二氧化硅
  • 像水，乙醇这种，虽然是共价键形成的物质，但它们是 分子晶体 因为其分子是独立出现的，分子间的作用是分子间作用力 （它俩主要是 氢键） 二氧化碳也是 分子晶体，其分子间作用力是：范德华力
  • \(AlCl_3\) 是 共价晶体，所以它的熔融状态不导电，所以没法电解这个制备铝 “离子” 与 “共价” 没有明确区分，随着电负性差异从大到小，会从离子到共价 过渡
  • 混合晶体：例如石墨，“分层” 层内，六边形网格，共价晶体，共价键连接 层间，不同的网格之间，是范德华力

晶胞，空间结构相关

需要一些空间想象力

晶胞，是：

  • 晶体的 “周期”，
  • 是一个三维的 “周期”，并且是 “最小正周期”
  • 想像晶体结构，就想像把晶胞不断重复之后的样子
  • 通常，会取出一个正方体形状的晶胞

正方体晶胞：

  • 体心，面心，棱心
  • “四角”：联系数学，立体几何知识 正方体 \(ABCD-A_1B_1C_1D_1\)，取 \(AC\) 和 \(B_1D_1\)，连接，形成的是 正四面体 这个结构很常见

晶胞中常出现 “半个” “1/4个” “1/8个” 原子：

  • 计算晶胞内原子的数量，就按分数计算
  • 晶胞内原子数目的比 = 整个晶体原子数目的比 比如，晶胞内 \(A,B\) 有 \(1.5,2.5\) 个 那说明这个晶体可以写成 \(A_3B_5\)

配位数：离一个原子最近的，其它原子的数量

  • 通常，试卷上画的，会把原子和最近原子用线连起来 但是这个要注意：要把晶胞重复一下之后，看看别的周期里面的
  • 如果没画，就用数学的立体几何的手段，做一下 有些其实比较直观，好想象
